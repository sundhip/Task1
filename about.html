Projects Overview

OrganXcell

Organ Matching & Transplant Management System

OrganXcell is a comprehensive digital solution for medical networks. It secure-stores donor credentials, runs matching algorithm searches using HLA criteria and blood groups, and updates the recipient queues. The project avoids manual paperwork bottleneck delays, preventing organ expiration risks.

Tech Stack: Python, Django, MySQL Database, HTML, CSS.

EvolveAura

Gamified Self Improvement Platform

EvolveAura gamifies personal productivity. Users set daily goals, log tasks, and earn experience points to unlock profile avatar modifications and unique status badges. Built to keep young learners motivated using classic role-playing-game (RPG) loop aesthetics.

Tech Stack: React.js, Node.js, Express, MongoDB.

Sensor Analysis Project

Glass & Plastic Detection using Capacitive Sensors

A robotics and materials sorting project. Utilizing physical microcontrollers, it processes dielectric changes from capacitive sensors to classify object materials. Useful in sorting recyclables for waste automation.

Tech Stack: C++, Arduino IDE, Embedded Electronics.
